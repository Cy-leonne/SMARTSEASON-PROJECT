# 🌱 Smart Season

Smart Season is a modern agricultural management platform designed to help admins and field agents collaborate seamlessly. It provides dashboards, crop tracking, and real-time updates to streamline agricultural operations.

---

## ✨ Features
- **Role-based authentication**: Secure login for admins and agents
- **Admin Dashboard**: Overview of fields, crop distribution charts, and detailed tables
- **Agent Dashboard**: Assigned fields view, update forms, and history tracking
- **RESTful API**: Endpoints for authentication, dashboard data, and field updates
- **Data Visualization**: Charts for crop status and growth stages

---

## 🛠️ Tech Stack
- **Frontend**: React, Axios, React Router, Recharts  
- **Backend**: Node.js, Express, JWT Authentication  
- **Database**: MongoDB  
- **Tools**: Postman for API testing
