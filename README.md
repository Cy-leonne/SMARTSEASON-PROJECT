# 🌱 Smart Season

**Smart Season** is a modern agricultural management platform designed to help admins and field agents collaborate seamlessly. Built with a **Node.js/Express backend** and a **React frontend**, it empowers users to track crop fields, monitor growth stages, and streamline communication between teams.

---

## ✨ Features
- **Role-based authentication**: Secure login and registration for admins and agents.
- **Admin Dashboard**  
  - Overview of total fields and most common crops  
  - Interactive charts for crop status distribution  
  - Field table with detailed information  
- **Agent Dashboard**  
  - Assigned fields view  
  - Update form to submit crop stage and notes  
  - Real-time refresh to see updates instantly  
- **History Tracking**: View past updates per field for better decision-making.
- **RESTful API**: Backend endpoints for authentication, dashboard data, and field updates.

---

## 🛠️ Tech Stack
- **Frontend**: React, Axios, React Router, Recharts  
- **Backend**: Node.js, Express, JWT Authentication  
- **Database**: PostgreSQL 
- **Tools**: Postman for API testing
